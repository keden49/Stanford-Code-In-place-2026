# Stanford-Code-In-place-2026

This repository contains all coursework, assignments, and projects completed during the 2026 session of Stanford University’s Code in Place. This six-week introductory programming course is based on the first half of Stanford’s CS106A curriculum and focuses on Python fundamentals through a human-centric learning model. The contents of this repo track my progress from learning logic with Karel the Robot to mastering core concepts like control flow, functions, and data structures, culminating in a final independent project.
